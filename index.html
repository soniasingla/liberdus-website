FAQ

Frequently Asked Questions

Here are some common questions asked about Liberdus.

What is Liberdus?

+

Liberdus is a decentralized messaging and payment platform that allows users to send secure messages and cryptocurrency payments in a single transaction. It combines the convenience of messaging apps with the power of blockchain technology.

What is Shardus?

+

Shardus is a distributed ledger technology that uses dynamic sharding to solve the blockchain trilemma of scalability, security, and decentralization. Liberdus is built on the Shardus protocol to achieve linear scalability.

What are the tokenomics for Liberdus?

+

LIB has a fixed supply of 210 million tokens with 18 decimals, following Bitcoin's proven economic model. The token is used for network fees, governance, and incentivizing network participants. No inflation, no dilution.

How can I run a node to earn rewards?

+

Node operators help secure the network and earn LIB rewards. During testnet, you can run validator nodes and earn rewards for participating in network consensus and validation. More details will be available when testnet launches.

How can I add liquidity to earn rewards?

+

You can provide liquidity to LIB trading pairs on decentralized exchanges like Uniswap. Liquidity providers earn trading fees and may receive additional LIB rewards through liquidity mining programs.

How do I support the project?

+

You can support Liberdus by contributing code on GitHub, joining our Discord community, providing feedback during testnet, creating educational content, or helping with translations and documentation.
